Сайт-визитка для компании, предоставляющей услуги грузоперевозок. Проект разработан в рамках университетского курса по веб-разработке. Позволяет клиентам оставлять заявки на перевозку грузов и связываться с компанией через форму обратной связи.

Английская версия доступна [здесь](README.md).

## Описание
Многие компании по грузоперевозкам не имеют структурированного онлайн-представительства, из-за чего коммуникация с клиентами происходит хаотично (телефоны, мессенджеры). Данный сайт решает эту проблему, выступая единым информационным ресурсом и инструментом первичного обращения. Клиенты могут заполнить заявку на перевозку или отправить сообщение через форму обратной связи. Менеджеры получают структурированные данные, что ускоряет обработку заявок и снижает количество ошибок.

## Используемые технологии
- **Frontend:** HTML5, CSS3, JavaScript (нативный)
- **Backend:** PHP (без фреймворков)
- **Базы данных:** MySQL (структурированные заявки), MongoDB (обратная связь и комментарии)
- **Инфраструктура:** Docker, Docker Compose
- **Безопасность:** переменные окружения (`.env`), валидация данных

## Возможности
- Форма подачи заявки на грузоперевозку (имя, контакт, детали груза, маршрут).
- Форма обратной связи для вопросов и комментариев.
- Информация о компании и услугах на главной странице.
- Полностью адаптивный дизайн (ПК, планшет, телефон).
- Двухбазовая архитектура: MySQL для заявок, MongoDB для гибких сообщений обратной связи.

## Установка и использование

### Требования
- Windows / macOS / Linux с установленными Docker и Docker Compose.
- Git.

### Шаги по установке
1. Клонируйте репозиторий:
   
  `git clone https://github.com/your-username/Site-for-truckers.git`
  
  `cd Site-for-truckers`
   
2. Скопируйте файл с переменными окружения:

  `cp .env.example .env`
   
  Отредактируйте .env, указав параметры подключения к базам данных и другие настройки.

3. Запустите Docker-контейнеры:

  `docker-compose up -d`

4. Инициализируйте базы данных:

- Импортируйте схему MySQL из database/mysql_schema.sql (если есть).

- MongoDB создастся автоматически при первом обращении.

5. Откройте сайт в браузере: http://localhost:8080.

### Как использовать

1. Изучите информацию об услугах на главной странице.
2. Заполните форму заявки на перевозку (имя, контакт, тип груза, маршрут).
3. Используйте форму обратной связи для отправки вопросов.
4. Ваши заявки и сообщения попадают напрямую в базу данных компании; менеджеры свяжутся с вами.

### Ограничения
- Проект учебный, не предназначен для промышленного использования.
- Нет личных кабинетов и истории заявок — все запросы обрабатываются менеджерами вручную.
- Административная панель минимальна; полноценная CRM планируется в будущих версиях.
